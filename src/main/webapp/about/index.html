Helpdesk Information

Welcome

This application was lovingly built by Galen Schmidt to serve as a signin application for the Rice University IT helpdesk.

Source Code

The application is set up as a Maven project. Technologies used include myBatis, Jersey, Java Servlets, and JAXBE on the backend, and AngularJS with twitter bootstrap on the frontend. If you're interested in looking at the source code, the entire project is on github

Visit github respository »

Ideals

This system was built to replace the previous one installed by Austin Whitt. Although the system he built met all of the requirements asked of it, it was difficult to maintain (and written in PHP!). The new system strives to be simple, powerful, and maintainable.
